About this project

This project is a basic implementation of a restaurant database where users can create restaurants, menus, update and delete them. I have made available an api that allows users to access to the restaurant list as well as menus available by using JSON endpoints


Get Menu Items for each restaurant:

In order to access to the list of menus for one restaurant you have to enter the following url: localhost:5000/restaurants/developers/restaurant-id/menu/JSON and will will receive a JSON formatted response.

Get each menu item separately:

In order to access to each menu separately you have to enter the following url: localhost:5000/restaurants/developers/restaurant-id/menu/menu-id/JSON

Get restaurant Information:

By entering the following url: localhost:5000/restaurants/developers/restaurant-id/JSON you will have access to the restaurant information such as name, email, phone number, description etc..

Get restaurant Location:

By entering the following url: localhost:5000/restaurants/developers/location/restaurant-id/JSON you will have access to the restaurant location including address, city, state and zip code..
